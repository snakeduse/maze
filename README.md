# Maze

A small 2D browser maze game built with Vite, TypeScript, and HTML Canvas.

## Install

```sh
npm install
```

## Run

```sh
npm run dev
```

## Build

```sh
npm run build
```

## Controls

- Move: WASD or Arrow keys
- Restart: R
- Reach the goal to complete the level.
- Avoid spikes.

## Map Symbols

- `#` wall
- `.` floor
- `P` player start position
- `G` goal
- `S` spikes
